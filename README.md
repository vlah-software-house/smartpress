# YaaiCMS

AI-driven content management system built with Go, HTMX, AlpineJS, and PostgreSQL.

> **Note:** This project was previously known as SmartPress. It has been renamed to YaaiCMS for trademark reasons.

## Tech Stack

- **Backend:** Go
- **Frontend:** HTMX, AlpineJS
- **Database:** PostgreSQL
- **Cache:** Valkey
- **E2E Testing:** Playwright
